Serialization & Deserialization (It is recommended to use if json payload is complex)

Serialization : It is process of converting pojo class object into json payload

Deserialization : It is process of converting json payload into pojo class object

pojo --> plain old java object

For Serialization & Deserialization we should add below tools in our project

	jackson databind
	jackson annotation
	jackson core 
	Gson
